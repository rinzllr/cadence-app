# Cadence

Smart habit tracking app with pattern detection and accountability.

## What It Does

Cadence helps you build better habits by:
- Creating smart reminders that only appear if you completed the previous one
- Capturing feedback and notes for habits you want to learn from
- Showing statistics and trends to identify patterns
- Tracking both completions and failures to understand your habits better

## Why Cadence?

Existing habit apps are either too simple or too complex. Cadence is straightforward, no-BS habit tracking focused on what actually matters: understanding your patterns and building consistency.

## Tech Stack

- **Backend:** FastAPI, PostgreSQL
- **Frontend:** React 18
- **Mobile:** React Native (Phase 2)
- **Deployment:** GCP, Vercel
